# Task 024.8: Use polygon union for avoid zones

## Goal

Merge overlapping Rural Area Finder circles with real polygon union logic.

## Why this matters

Task 024.7 grouped overlapping circles but rendered multi-building groups as convex hulls. That was not the expected behavior: overlapping circles should combine into one shape that follows the merged circle outlines, not a stretched hull around them.

## Implementation steps

1. Add a small polygon clipping/union dependency.
2. Convert each approximate circle ring into polygon input.
3. Union polygons within each overlapping group.
4. Emit the union result as map features.
5. Build the app.
