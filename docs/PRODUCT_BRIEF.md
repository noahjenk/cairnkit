# CairnKit Product Brief

## Product summary

CairnKit is a modular, browser-based scouting map platform for finding, saving, and organising interesting places.

It opens directly into a map. The app should feel practical, outdoorsy, exploratory, flexible, and useful.

## Tagline

A modular map toolkit for finding and saving places.

## Brand meaning

- **Cairn** means an individual saved or marked location.
- **Kit** means the wider set of modular map tools.

## First major tool

The first major tool is **Rural Area Finder**.

Rural Area Finder helps users identify areas that are farther away from selected mapped features, starting with buildings.

Rural Area Finder is one tool inside CairnKit. It should not become the whole app.

## Product principles

1. Map-first interface.
2. Modular tools.
3. Layer-based map output.
4. Adapter-based data sources.
5. Adapter-based storage.
6. Local-first saved places.
7. Beginner-friendly codebase.
8. Documentation as source of truth.
9. Small testable implementation tasks.
10. No large unplanned rewrites.

## MVP includes

- Full-screen MapLibre map.
- CairnKit app shell.
- Top bar.
- Left workspace panel.
- Search box placeholder.
- Status indicator.
- Tool system.
- Layer system.
- Rural Area Finder shell.
- Mock building data source.
- Radius slider.
- Approximate avoid-zone layer.
- Saved places stored locally.
- All Places list.
- Saved-place pins on the map.

## MVP does not include

- Accounts.
- Login.
- Cloud sync.
- Photos.
- Tags.
- Social sharing.
- Project or trip grouping.
- Advanced drawing tools.
- Full mobile app experience.
- Offline mode.
- Production backend.
- Perfect GIS processing.
- Polished dissolved buffer algorithm.
